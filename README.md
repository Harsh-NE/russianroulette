1. Clone the repository
git clone https://github.com/Harsh-NE/russianroulette.git
cd russianroulette
2. Install dependencies

Backend:

cd backend
npm install

Bot Engine:

cd ../bot-engine
npm install

Frontend:

cd ../frontend
npm install
3. Run the project

Start backend:

cd backend
npm start

Start bot engine:

cd ../bot-engine
npm start

Start frontend:

cd ../frontend
npm start
